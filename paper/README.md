# Paper status

`draft.md` is preserved as a historical research draft. Its complete raw
Rostam attempt archive was not retained, so the current repository cannot
regenerate or independently validate its numeric claims. It is intentionally
excluded from CommCanary release distributions under
[ADR 0008](../docs/adr/0008-paper-publication-boundary.md).

New publication material must come from the manifest-driven experiment
pipeline: immutable attempts, explicit selection, a complete verdict, and the
canonical `aggregate.json`, `aggregate.csv`, and `paper-fragment.md` emitted by
`experiments/rostam/analyze.py`. No hand-copied headline value is a substitute
for those generated bytes.
